⚙️ Supabase Setup

To enable shared checkout tracking, you need a free Supabase account. Follow the steps below.

Step 1 — Create a Supabase project

  1. Go to supabase.com → Sign up free
  2. Click New project
  3. Name it mkpc-library
  4. Set a database password (save it!)
  5. Wait ~2 min for it to provision

Step 2 — Create the checkouts table

  1. In your project, go to SQL Editor
  2. Click New query
  3. Paste the SQL below and click Run
CREATE TABLE checkouts (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  book_id integer NOT NULL,
  borrower_name text NOT NULL,
  phone text,
  due_date date,
  notes text,
  checked_out_at timestamptz DEFAULT now(),
  returned_at timestamptz
);
ALTER TABLE checkouts ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow all" ON checkouts FOR ALL USING (true) WITH CHECK (true);

Step 3 — Get your API credentials

Go to Project Settings → API. Copy your Project URL and anon/public key and paste them below.
